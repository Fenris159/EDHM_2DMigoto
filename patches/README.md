# EDHM patches

Place EDHM-specific notes, patch descriptions, and exported diffs here.

Suggested layout as work accumulates:

```text
patches/
  README.md
  0001-example-compat-fix.md
  notes/
```

Each patch note should record:

- Problem (EDHM symptom)
- Root cause (if known), ideally with comparison to classic 3Dmigoto
- Upstream SHAs considered (`xxmi` / `3dmigoto`)
- Local commit(s) that implement the fix
